# GitScope 0.1.0

GitScope's first public release turns scoped GitHub organization activity into a
private, evidence-based engineering career report.

## Highlights

- Collect authored commits, pull requests, reviews, and issues from an explicit repository allowlist.
- Match historical Git author names and email aliases across account changes.
- Analyze complete Git history using reusable full bare mirrors.
- Explore repository, timeline, language, file-extension, pull-request, review, and issue analytics.
- Generate a responsive offline HTML dashboard, Markdown report, CSV ledger, JSON contract, and standalone charts.
- Produce synchronized Markdown and HTML contribution résumés with light, dark, and print themes.
- Regenerate every derived output offline from an existing `report.json`.
- Inspect local health and manage regenerable cache content without displaying private payloads.

## Installation

Install the wheel attached to this release:

```bash
uv tool install \
  https://github.com/CaptainOfFlyingDutchman/gitscope/releases/download/v0.1.0/gitscope-0.1.0-py3-none-any.whl
```

Then verify the command:

```bash
gitscope --version
gitscope doctor
```

GitScope requires Python 3.13 or newer and Git. It is distributed through this
GitHub release, not PyPI. The `gitscope` project currently found on PyPI is
unrelated.

## Privacy

GitScope processes private organization information locally. Review reports
before sharing them, use least-privilege GitHub access, and follow the repository
[security guidance](https://github.com/CaptainOfFlyingDutchman/gitscope/blob/v0.1.0/SECURITY.md).

Every attached distribution is built and installed in isolation before release.
Use `SHA256SUMS` to verify downloaded assets.
