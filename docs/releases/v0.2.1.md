# GitScope 0.2.1

GitScope 0.2.1 improves chart readability and makes zero-issue reports more
explicit.

## What changed

- Added opaque, high-contrast Plotly hover-label backgrounds, borders, and text.
- Added matching hover-label colors for light and dark dashboard themes.
- Kept the Issue Outcomes card visible when no authored issues were collected.
- Added a clear empty-state message instead of rendering an unexplained blank chart.

## Installation

Install the wheel attached to this release:

```bash
uv tool install \
  https://github.com/CaptainOfFlyingDutchman/gitscope/releases/download/v0.2.1/gitscope-0.2.1-py3-none-any.whl
```

Existing GitScope tool installations can upgrade with:

```bash
uv tool install --force \
  https://github.com/CaptainOfFlyingDutchman/gitscope/releases/download/v0.2.1/gitscope-0.2.1-py3-none-any.whl
```

GitScope is distributed through GitHub Releases, not PyPI. Use `SHA256SUMS` to
verify downloaded assets.
