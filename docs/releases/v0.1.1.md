# GitScope 0.1.1

GitScope 0.1.1 is a focused visual polish release for its portable HTML outputs.

## What changed

- Added a crisp GitScope SVG favicon using the dashboard's navy-and-blue visual language.
- Added the favicon to the offline career dashboard and contribution résumé.
- Added the favicon to every standalone Plotly chart page.
- Kept the asset fully local so generated reports remain private and work without network access.
- Extended distribution and installed-wheel verification to protect favicon packaging.

## Installation

Install the wheel attached to this release:

```bash
uv tool install \
  https://github.com/CaptainOfFlyingDutchman/gitscope/releases/download/v0.1.1/gitscope-0.1.1-py3-none-any.whl
```

Existing GitScope tool installations can move to this release with:

```bash
uv tool install --force \
  https://github.com/CaptainOfFlyingDutchman/gitscope/releases/download/v0.1.1/gitscope-0.1.1-py3-none-any.whl
```

GitScope requires Python 3.13 or newer and Git. It is distributed through this
GitHub release, not PyPI. The `gitscope` project currently found on PyPI is
unrelated.

Every attached distribution is built and installed in isolation before release.
Use `SHA256SUMS` to verify downloaded assets.
